The data presented here were obtained and described in the following publication:

"Global signals of the first molecules from the dark ages in the presence of primordial magnetic fields" 

https://doi.org/10.1103/92bl-7pcx

https://doi.org/10.48550/arXiv.2510.10212

Data files contain headers. To build the graphs in the publication, use file grs.tex and comands

latex --jobname=g2 grs.tex

dvips g2.dvi

dvipdf g2.dvi

Please note that the data files must first be placed in subfolder 'f'.
